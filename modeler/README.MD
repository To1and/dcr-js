## DCR graph diagram modeler

Now part of the refactored dcr-js project. The modeler has been copied over into the refactored app. Touch at your own peril

# DEPRECATED

## Development

### Installing dependencies
```console
npm i
cd starter
npm i
```

### Running the modeler
```console
cd starter
npm run dev
```
